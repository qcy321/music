专辑介绍：


这城市

太多形色匆匆的人

有太多事情要做……

那追逐 那较量 那欲望

是在梦里都放不下的我执

谁都自以为看透了现世

但谁又真的想离开

如果白昼注定以奔忙来填满

梦中就请以畅游来救赎

邀你参与，一场精心策划的出离。

——2012/7/11 · 许嵩第四部作品「梦游计」

浮华的现代都会，人与人、事与事之间千丝万缕的纠葛，使得白天总在上演着一场场追逐欲望的脑力较量。但回归到一个人的深夜，可否大胆借梦游一场，作为逃离一切束缚的妙计？在梦游的领域，抛开压力与顾虑，站在自己之外看自己，重新审视自己与他人、自己与世界的关系——超现实主义作品《梦游计》将提供这样的可能性。作者以敏锐的洞察力，天马行空的想象，出其不意的运笔，自成一派的音乐理念，邀你完成一场梦游式的出离。

第四部作品《梦游计》，许嵩依然包揽十首词曲的创作、担当制作人及唱片统筹，用一整个25岁锤炼独立音乐作品新范本，再一次展现了惊人的音乐能量。后期环节中携手许环良、林哲民、蔡庭贵、杨阳、陈伟伦等一线配乐人与混音大师，在他的工作室磨合切磋了上百个日夜，精益求精，只为上演完美“梦游计”。视觉方面，平面写真融入超现实主义元素，以虚实相生的表现手法呈现“梦游”情境；MV奔赴台湾打造专属于《梦游计》的梦游诡谲风格；比尔贾、黄中平、方亮等众多视觉创意高手前来加盟，联手奉上独具梦游滋味的眼球盛宴。你将领略这样的许嵩，嬉皮又文雅的，迷离又真实的，戏谑感十足却充满诚意的。

许嵩说，整部作品的创作概念本就源于一场梦。他在那梦里游走，超脱于现实，却得以更真实的触碰现实；他把他所历经的一切，摊开来给你看。

＊ 全碟十首作词+作曲+制作，许嵩第四次展现强大音乐能量

＊ 2009-2012连续发表四张词曲全创作专辑，笔耕不辍，佳作连连

＊ 2011年凭《苏格拉没有底》成为内地唱片销量王，获得国内外共计9项音乐大奖

如果你对他的音乐还不了解，那么《梦游计》是一场很好的初见；如果你是他的音乐追随者，那么《梦游计》将第四次印证你的选择。

＊单曲介绍

01. 胡萝卜须

法国黑色童话《胡萝卜须》里的那个红头发少年，盯着自己亲手摔碎的鼹鼠，任由它的血花在空中飞扬……

你未必没有暗暗释放过——扭曲的、荒诞的、隐藏在你道德外衣下的暴力。

因为你一定暗暗接收过——傲慢的、偏狭的、让你愤懑却也令你成长的眼神。

谁 可记取几桩童年旧梦

谁 又深藏几段残破青春

谁 能说自己不是胡萝卜须

《胡萝卜须》由许嵩作词、作曲及制作，是他第四部独立作品《梦游计》的开篇序曲。作为大陆词人中的新锐标杆人物，许嵩向来以笔法的大胆与视角的独特见长。此次新歌以法国文学名作《胡萝卜须》的主人公为切入点，运笔穿梭于梦样童话与冰冷现实之间，在“自我”与“胡萝卜须”两端来回切换，令人读罢笑中有泪；音乐方面，许嵩创作极具西洋色彩的半音阶旋律线，并创意性的融入探戈节奏，在手风琴与萨克斯的呼应之下自然而然地散发出异域风情；标志性的“懒人唱腔”此番亦演绎至极致，梦游般慵懒而又充满戏谑的独特诠释，令《胡萝卜须》的黑色幽默在短短的三分多钟里无限扩张。词、曲、唱、制的四者合一，给他的作品打上了鲜明的个人烙印。知名导演比尔贾掌镜MV，以寓意巧妙的视觉语言展现胡萝卜须的精神世界，打造出专属于《梦游计》的超现实主义诡谲风格。

02. 幻听

「 在远方的时候 / “又想你到泪流” / 这矫情的措辞结构 / 经历过的人会懂 」

你是否幻听过你的前度在你耳边低语？

当你发现那不过是一场幻觉，不过是自己大脑制造出的假相——你是否仍还希望那假相能持久一些？

很多时候，回归现实真不如躺在幻觉里永不醒来……

真诚质朴的遣词造句，精致纯美的旋律，感性细腻的演绎，给人带来最原始的感动。配乐方面，一把吉他、几声鼓点，淡淡的抒情慢板R&B曲风，纯粹到极致，也深情到极致；知名MV导演黄中平与许嵩再度合作，以幻梦风格呈现他笔下的这场“极致爱情”。

“如果一个人被魔鬼深深凝视过了，那么他的眼睛就会瞎掉。

如果一个人被这首歌轻轻带入了，他的耳朵会幻听。”

03. 对话老师

「 我写检讨的文采不到写情书的一半 / 你做学问的本领有没五柳先生一半 」

淘气的闯祸，早恋的情愫，焦急盼望的下课声，害怕家长发现的不及格考卷……念书时发生过很多让自己脸红的故事吧？

但，那不正是长大后的心中最美好的那些年？

许嵩梦回学生时代，信手拈来一曲校园小品，幽青春一默，也幽老师们一默。

04. 伴虎

「 谁能以为 / 同朝可同寐 」

历史的车轮碾过，多少英雄马到功成，多少亡魂哀怨嗟叹？

功高震主的人，杀身之祸总是难免。

而对于“主子”而言，事后文过饰非颠倒黑白，比起杀戮来，更是拿手好戏。

至于仅存在于少数人心中的真相，随着时间推移，终会飘散无踪……

许嵩全新创作中国风，跳出当下一切流行套路。梦游旧朝，明写君臣关系暗抒历史观，意旨深远。如果说作品前半段还是他标志性的慵懒与忧伤，后半段则令人拍案惊喜：多把失真吉他的层叠渐进，将格局骤然扩张；撕裂式的高音演唱，彻底抛开声乐传统；第四分钟之后甚至飚出男高音真声罕有企及的超高音“Hi C#”，将悲悯情绪发泄到极致。MV导演陈映之为让画面体现歌曲深沉质感与磅礴大气，大手笔启用直升机航拍，壮阔大海、层峦崖石、平沙落日、幽秘森林，一一全景式呈现。

05. 闺蜜

「 闺蜜一出手 / 真让你难受 / 现实教会了人要往前游 / 却弄丢朋友 」

“闺中密友”不是应当简作闺密吗？为何新新人类惯用“闺蜜”？

可能换了个“蜜”字，就像是能增加几分同甘共苦的甜蜜感吧。

可许嵩笔下的这对闺蜜，有不同的故事。小小说般层层推进情节，用词精炼老道；结尾话锋一转，升级立意。配乐怀旧却又带着鲜明的节奏律动，这是专属于他的音乐特质。

他写闺蜜关系，君臣关系，师生关系，……他度量着人与人之间的温度，用他温暖的腔调捂热现实的冰冷。

06. 装糊涂

「 对于你的谎话我装糊涂 / 装糊涂是极难的艺术 / 稀里糊涂是我的包容 」

在爱情里，一切的质疑都是不必要的。因为，如果不能接受，干脆趁早分开；如果不想分开，那就装糊涂吧，不然就算鱼死网破得到了真相，又能有什么更好的结果？

——但要小心，某一个清晨醒来也许会发现，自以为的温存竟是梦游般的幻觉，现实中早已一无所有！

装糊涂，装到真的糊涂了……

许式经典R&B，逐层递进的配唱方式，摇摆感性的节奏律动，把这首歌曲推送到每个“糊涂着却不糊涂”的人心中。2012暑期档影片《影子富豪》制片方在第一次听到歌曲小样时，就立马邀此歌作为电影主题曲。

07. Play with style

何谓Play with style？抛开潮流的诱惑，突破名相的束缚，活出真自我，找到自己的态度是为先！此作品除了歌词层面传达了这一意旨，音乐方面也极具先锋性：世界民族乐器的大量融入，给音乐增添了一份神秘的色彩——但却又辅以西洋感的R&B鼓点、极前卫的弹性演唱方式——这种古今碰撞所产生的特殊质感，恐怕只有许嵩能独家定制。

08. 心疼你的过去

「 忽然你哭了 / 问你在哭什么 / 你哭的更狠了 / 我好像明白了 」

每个人都有过去，但不是每一段过去都能轻易过得去。

你若爱Ta，你便很可能纠结于Ta的过去；Ta若爱你，便很可能对你隐瞒Ta的过去。

如果时间是一条河……你是否愿意在梦里游回到岁月上游，去更早的时间点和Ta相遇，为Ta拦下所有的伤害？

许嵩此作采用日记体写作风格，整篇没有任何诗意斧凿，用最真实的笔触把最纠结的心疼刻画到血肉里，令人惊心。循环渐进式的R&B Loop编曲与深情款款的念唱，逐层铺排出幽诡阴郁的氛围。从《伴虎》的激越磅礴，到《对话老师》的顽皮十足，再到《幻听》的传统深情，直到这一首的Rap……许嵩作为创作人以及演唱者，在《梦游计》这张专辑中展现了对多种曲风的极佳驾驭力，使得此专辑一经推出无论在广度还是深度上都备受瞩目。

09. 全球变冷

这是一个全球化的时代，这是一个地球村。但不知道从什么时候开始，繁盛的背后培育起冷漠，GDP的上扬伴随着道德的滑坡，丰富的物质世界背后是精神世界的逐渐荒芜……我们是该坐视人与人之间冰山横陈、任凭全球变冷，还是用微笑温暖彼此？许嵩此作写于“小悦悦”事件之时，不为问责，只愿同样的悲剧能少一些，人与人之间的关怀能多一些。

10. 亲情式的爱情

「 我有多想多想再看一眼 / 你最后那天在我身边 / 那熟悉的脸 ……」

是否电光石火般的爱恋，最终都会归于平淡？当曾经的日思夜想辗转反侧魂不守舍，沦为一起吃饭睡觉买菜吵架做家务，恐怕就再也没有给对方写一封情书的雅兴了……

但，爱是消失了，还是悄悄转化了存在形式？

许嵩这首作品的创作，是感于新闻里一件真人真事。一对中年夫妻遭遇了一场严重车祸，丈夫卡在车轮下难以救出，他停止心跳前对救援人员的最后一句话是，别管我，先救我爱人！说完便再也没有睁开眼睛。侥幸生还的妻子在事后哭着接受采访说，就在车祸发生前一分钟，他们还在为家庭琐事吵嘴，吵得很凶，而前一阵还在闹离婚。……

珍惜身边人，让爱从日常生活里流淌出来，不要等到失去才追悔，一辈子并没有想象之中那么长。
